# AGT E Motors - Site Web

Ce projet est un site web pour AGT E Motors. Suivez les étapes ci-dessous pour installer et exécuter le site en local.

## Prérequis

Avant de commencer, assurez-vous d'avoir installé les éléments suivants sur votre machine :

- [XAMPP](https://www.apachefriends.org/index.html) (Pour le serveur Apache et MySQL)
- [Composer](https://getcomposer.org/) (Gestionnaire de dépendances PHP)

## Étapes d'installation

### 1. Installer XAMPP

Téléchargez et installez XAMPP sur votre machine depuis le [site officiel d'Apache Friends](https://www.apachefriends.org/index.html).

Une fois installé, ouvrez XAMPP et lancez les services **Apache** et **MySQL**.
